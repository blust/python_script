## 1. dnspod动态提交域名解析记录(dnspod.py)
经常想把家里的电脑作为服务器绑定到外网使用, 但是ip是动态的, 这个脚本就是借助dnspod的解析api, 自动把域名解析到家里电脑的公网ip
然后从路由器开启dmz主机, 就可以在外网用域名访问家里的电脑了
具体教程点击 [dnspod脚本使用教程](http://www.phpue.com/python/4.html)
